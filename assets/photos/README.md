# Photos

Place lab photos, group photos, profile pictures, and research visualizations in this folder.

Recommended naming:

- `group-photo-2026.jpg`
- `prof-ravindran.jpg`
- `member-name.jpg`
- `heusler-lattice-visualization.png`

Use compressed `.jpg` or `.webp` files for photographs and `.png` for diagrams that need crisp lines.
